## Core Features

- WhatsApp assistant with permissioned chats and QR pairing
- Slack assistant with channels, alerts, and slash commands
- MCP servers for tool integrations (Jira, Google, etc.)
- Scheduler for recurring and one-off jobs
- Mini-Apps builder and app hosting
- Agent registry for configuring skills and tool permissions
- Database-backed secrets management (API keys, tokens)

## Dashboard Areas

- WhatsApp: configure chats, permissions, pairing status
- Slack: manage channels and permissions
- Integrations: MCP servers and dual-mode settings
- Agents: registry of agent configurations
- Schedules: recurring and one-off jobs
- Apps: mini-apps management
- Secrets: manage API keys and tokens securely in the database
- Billing: track API usage and costs
