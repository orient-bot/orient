## What is Orient?

Orient is a powerful AI-powered automation platform that integrates with your messaging platforms to help you:

- **Ask questions** and get intelligent answers
- **Schedule messages** and set reminders
- **Create mini-apps** like forms, schedulers, and dashboards
- **Manage tasks** and track progress
- **Integrate with tools** like Jira and Google Workspace

## Getting Started

Choose your preferred platform to get started:

- **[WhatsApp](./getting-started/whatsapp)** - Connect via QR code or pairing code
- **[Slack](./getting-started/slack)** - Add to your workspace

## Self-Hosting

Orient is open-source and can be self-hosted. See our [GitHub repository](https://github.com/orient-bot/orient) for setup instructions.

Quick start for developers:

```bash
# Check prerequisites
./run.sh doctor

# Install and start
pnpm install
./run.sh dev
```
