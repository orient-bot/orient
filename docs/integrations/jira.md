# Jira Integration

## Requirements

- Jira Cloud site
- API token for your user

## Setup

Add these to `.env`:

- `JIRA_HOST`
- `JIRA_EMAIL`
- `JIRA_API_TOKEN`
- `JIRA_PROJECT_KEY`
- `JIRA_COMPONENT` (optional)

## Notes

Project key and component are organization-specific and should not be hardcoded.
